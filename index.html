Advanced-Sidebox

A plugin for MyBB forums that displays portal boxes on various forum pages.

Download this project as a .zip file Download this project as a tar.gz file

Advanced Sidebox 1.3.7 for MyBB 1.6.x

This plugin allows the forum admin to add and manage side boxes on the forum in a intuitive way and is packed with powerful options. This plugin has seen many improvements and thanks to a lot of community support has grown more powerful over time.

The Future

Advanced Sidebox 1.4 (beta) is currently in testing and is close to release.

Feature Changes:

  • Control which user groups side boxes are displayed for
  • Individual box settings rather than multiple boxes of the same type sharing settings
  • Simplified module management
  • Added functionality to automatically upgrade modules seamlessly saving the admin (and add-on module developer) work and time
  • Internal changes that will result in better performance and increased overall power.
